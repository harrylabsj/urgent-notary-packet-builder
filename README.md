# Urgent Notary Packet Builder

Build a same-day notary readiness packet with document inventory, ID checklist, signer logistics, and verification prompts — while avoiding legal advice.

## What it does
- Creates a document inventory with page counts, copies, and signature-block checks
- Builds a signer ID checklist covering expiration dates, name mismatches, and accessibility needs
- Confirms witness requirements, appointment logistics, and payment/pickup plans
- Produces a ready-to-bring checklist with questions for both the notary and receiving institution

## Example scenarios

**1. Same-day court affidavit notarization**
> 👤 "I need a notarized affidavit for a court filing by 5 PM today. Here's the unsigned document."
> 🤖 Inserts notary block template matching the jurisdiction. Adds signer-instruction highlights (don't sign until before notary, bring 2 forms of ID). Locates 3 nearby notaries with current hours.

**2. Generate a notary journal entry for records**
> 👤 "Generate a notary journal entry template for my records after the signing."
> 🤖 Produces a journal-entry PDF capturing date, document type, signer name, ID presented, signature, thumbprint placeholder, and fee paid.

**3. 出国留学紧急学位公证**
> 👤 "要出国留学，对方学校突然要求做学位公证，下周就是截止日了。我在上海，最快怎么搞定？"
> 🤖 紧急公证流程：随申办App在线预约加急→准备学位证+学历证+成绩单原件+翻译件（公证处合作翻译公司加急）→选有涉外业务的分处（大成/东方公证处）→加急1-2个工作日取件→快递到家。建议学位+成绩单+出生证明+无犯罪记录一次办齐。
